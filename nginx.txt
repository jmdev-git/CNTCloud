Generate a Self-Signed SSL Certificate

Download:
nginx - https://nginx.org/en/download.html
Stable version
nginx/Windows-1.30.0

EXE
openssl - https://slproweb.com/products/Win32OpenSSL.html
Win64 OpenSSL v4.0.0 Light

nssm - https://nssm.cc/download
Featured pre-release
nssm 2.24-101-g897c7ad

SSL FOLDER:
File:
cert.pem

Role:
The "Public" part. This is what you'll point to in Nginx as 
ssl_certificate or in Next.js as cert.

Typical Config Name:
fullchain.pem or cert.pem

File:
key.pem

Role:
The "Private" part. Never share this. Point to this as 
ssl_certificate_key or key.

Typical Config Name:
privkey.pem or key.pem

Path: C:\nginx\nginx.exe

Startup directory: C:\nginx (Do not leave this blank!)


Why the .\ is required
In Windows PowerShell, the current folder is not included in the "Path" for security reasons. This prevents malicious scripts from tricking you into running them just by being in the same folder. Adding .\ tells Windows: "I know this file is right here, run it anyway."

.\nssm.exe install Nginx
Service name: Nginx
